# barcode-broadcast-receiver

Native module for harware based scanners

## Installation

```sh
npm install barcode-broadcast-receiver
```

## Usage

```js
import { multiply } from 'barcode-broadcast-receiver';

// ...

const result = await multiply(3, 7);
```
